# SpringProject

A basic Spring Boot project using MongoDB.

## Git Commands Used

````
git init                            # Initialize the repository
git add .gitignore                  # Add the .gitignore file
git add .                            # Add all files
git commit -m "message"            # Commit changes
git remote add origin <URL>        # Add remote origin
git remote -v                      # Verify the remote URL
git branch -M main                 # Rename current branch to main
git pull origin main --rebase     # Pull remote changes with rebase
git rebase --continue              # Continue rebase if needed
git push origin main                  # Push to remote repository
